# Guitar Pattern Editor

Interactive fretboard tool for exploring scales and patterns on **guitar and bass**.

## 🎸 Try the App

[![Open App](https://img.shields.io/badge/Open-App-green?style=for-the-badge)](https://micatgthb.github.io/Guitar-Pattern-Editor)

## Features

- Guitar fretboard
- Bass fretboard
- Select root note
- Select scale
- Custom fret range
- Interval / note display
- Click notes to build patterns

## Usage

1. Select **instrument**
2. Choose **root note**
3. Choose **scale**
4. Adjust fret range if needed
5. Click notes to build patterns
